# Workout Remixer - Final Project

## 🚀 Overview
A high-performance fitness dashboard built with **FastAPI** and **Vue.js**. This application allows users to filter exercises by muscle group and difficulty, while using a custom "Remix" algorithm to find alternative movements.

## 🛠️ Technical Accomplishments (Solo Contribution)
* **Database Management**: Designed the SQLModel schema and implemented a custom seeding utility for automated deployment.
* **DevOps Pipeline**: Managed the full CI/CD lifecycle, deploying the containerized application to Render.
* **Remix Algorithm**: Developed the backend logic to query and suggest similar exercises based on muscle group parity.
* **Frontend Reactivity**: Built a dynamic UI using Vue.js that handles real-time filtering and asynchronous API updates.

## 💻 Tech Stack
* **Backend**: Python, FastAPI, SQLModel
* **Frontend**: Vue.js 3 (SFC-style), Bootstrap 5
* **Infrastructure**: Render, GitHub, SQLite
